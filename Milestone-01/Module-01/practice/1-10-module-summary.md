# 📚 Module 1 Revision

## 📌 HTML Semantic Tags

| Tag         | Purpose                                                      |
| ----------- | ------------------------------------------------------------ |
| `<header>`  | Represents the introductory section of a webpage or section. |
| `<nav>`     | Contains navigation links.                                   |
| `<main>`    | Holds the primary content of the webpage.                    |
| `<section>` | Groups related content into sections.                        |
| `<article>` | Represents an independent piece of content.                  |
| `<aside>`   | Contains supplementary or sidebar content.                   |
| `<footer>`  | Represents the footer of a webpage or section.               |

---

## 📌 Text Formatting Tags

| Tag        | Purpose                                  |
| ---------- | ---------------------------------------- |
| `<b>`      | Bold text (without semantic importance). |
| `<strong>` | Indicates strong importance.             |
| `<i>`      | Italic text (without semantic emphasis). |
| `<em>`     | Emphasized text.                         |
| `<mark>`   | Highlights text.                         |
| `<small>`  | Displays smaller text.                   |
| `<sub>`    | Subscript text.                          |
| `<sup>`    | Superscript text.                        |
| `<del>`    | Deleted text.                            |
| `<ins>`    | Inserted text.                           |
